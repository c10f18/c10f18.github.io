# clipboard 모듈 import
```
import clipboard
```

# clipboard 모듈 사용법
## 클립 보드에 복사
```
import clipboard

# 복사
clipboard.copy()
```

## 클립 보드에 붙여넣기
```
import clipboard

# 붙여넣기
clipboard.paste()
```
